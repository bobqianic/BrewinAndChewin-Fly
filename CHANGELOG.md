# 4.4.2-fly

Initial Brewin' And Chewin' Fly release for Minecraft 1.21.10.

- Added Create-Fly compatibility through the Create-Fly Modrinth artifact.
- Wine can be drunk directly from barrels.
- Recalculated satiety and saturation values for better food balance.
- Fermentation tanks can hold any liquid, and their temperature now changes gradually.
- Added large kegs.
- Added more realistic keg steam and cold-air particles.
- Empty kegs can be ignited, while heated liquid-filled kegs will evaporate.
- Fixed several coaster issues, including texture corruption, item stacking, and missing collision boxes for 3D models.
